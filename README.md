### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/spencerlore/vehicle-search-project
```

Go to the project directory

```bash
  cd vehicle-search
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
